# enagramJS Overview

enagramJS is an interactive, modular JavaScript learning platform that uses spaced repetition and a knowledge graph to help users master core programming concepts. The app is built with React and Vite, and is designed for extensibility and maintainability.

## Key Features
- Modular curriculum with unlockable topics
- Spaced repetition for long-term retention
- Interactive code challenges with instant feedback
- Progress tracking, XP, and streaks
- Modern, distraction-free UI

## Project Structure
- `src/components/`: Reusable UI components
- `src/utils/`: Utility logic (curriculum, spaced repetition, code execution, storage)
- `src/EngramApp.jsx`: Main application logic and views
- `tests/`: Dedicated test files
- `docs/`: Project documentation

## Getting Started
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev` or `make start`
3. Open http://localhost:5173/ in your browser
